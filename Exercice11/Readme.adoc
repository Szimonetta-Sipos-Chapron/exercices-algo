= Exercice 11 : Remplir un tableau

== Description

Écrivez un programme pour déclarer et initialiser un tableau, puis saisissez ses éléments à partir de l'utilisateur et affichez le tableau.

== Prérequis

* Les variables
* Les conditionnelles
* La fonction prompt
* La boucle for

== Déroulement

Données d'entrée 
Saisir le nombre d'éléments : 5
Elément 1 : 3
Elément 2 : 7
Elément 3 : 4
Elément 4 : 9
Elément 5 : 8

Données de sortie
3 7 4 9 8

== Attendus

* *Essentiel* : 1h 
* *Attendu* : 15 min
* *Avancé* : 5 min
