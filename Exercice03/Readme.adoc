= Exercice 03 : Pair ou impair

== Description

Écrivez un programme pour vérifier si un nombre est pair ou impair en utilisant if-else.

== Prérequis

Avoir des connaissances sur :

* Les variables
* Les conditionnelles
* La fonction prompt

== Déroulement

Données d'entrée :

2

Données de sortie :

Pair
