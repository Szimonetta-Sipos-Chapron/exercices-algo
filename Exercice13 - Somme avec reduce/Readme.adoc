= Exercice 13 : Somme d'un tableau

== Description

Écrire un programme pour déclarer un tableau, puis saisir ses éléments à partir de l'utilisateur et trouver la somme des éléments du tableau

== Prérequis

* Les variables
* Les conditionnelles
* La fonction prompt
* La boucle for

== Déroulement

Données d'entrée 
Saisir le nombre d'éléments : 5
Elément 1 : 3
Elément 2 : 7
Elément 3 : 4
Elément 4 : 9
Elément 5 : 8

Données de sortie
la somme = 31
